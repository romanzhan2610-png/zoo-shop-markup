# Проект "Дружок" — Верстка интернет-магазина зоотоваров

Проект представляет собой полностью адаптивную, семантическую верстку многостраничного интернет-магазина с реализацией сложной UI/UX логики на стороне фронтенда (интерактивные прототипы).

**Технологии:** HTML5, CSS3 (CSS Variables, BEM, Flexbox/Grid), JavaScript (Vanilla ES6+), Swiper.js, Yandex Maps API.

## Инструкция для тестирования интерактивных элементов

В проекте реализована имитация ответов сервера (mock-логика) для демонстрации пользовательского пути (User Flow).

### 1. Модуль авторизации и регистрации

_Открывается при клике на иконку пользователя в шапке._

- **Валидация (ошибки):** Отправка пустой формы подсвечивает поля красным. Ввод некорректных данных (например, логин `123`, пароль `123`) вызывает анимацию загрузки (лапка) и отображает глобальную ошибку «Неверный логин».
- **Успешный вход:** Для успешной авторизации необходимо ввести логин `test@mail.ru` и пароль `123456` (окно закроется после лоадера).
- **Восстановление пароля:** На этапе «Забыли пароль?» необходимо ввести `test@mail.ru`.
- **Ввод проверочного кода:** Правильный код для тестирования — `1111` (открывает экран создания нового пароля). Любой другой код выдаст ошибку.
- **Создание нового пароля:** При совпадении двух паролей отобразится финальный экран успеха с зеленой галочкой.

### 2. Выезжающая корзина и промокоды

_Открывается по клику на иконку в шапке или кнопки «В корзину» в каталоге._

- **Внутренняя логика:** Внутри корзины работает удаление товаров, изменение количества и добавление в избранное.
- **Промокод:** Для тестирования успешного применения скидки необходимо ввести промокод `TEST` или `SALE`. Это вызовет пересчет суммы «Итого» и появление строки скидки. Любое другое значение выдаст ошибку.
- **Бесшовный переход:** При нажатии «Перейти к оформлению» интерфейс корзины плавно сменяется предложением «Войти / Продолжить как гость».

### 3. Поиск

При вводе текста в главную строку поиска реализована имитация задержки (debounce-эффект): появляется лоадер, контент затемняется, после чего отображаются результаты поиска (категории и товары).

### 4. Карта контактов

Раздел контактов интегрирован с Яндекс Картами. При клике на карточку филиала в левом списке, карта плавно центрируется на соответствующем адресе. Используются кастомные SVG-маркеры.
